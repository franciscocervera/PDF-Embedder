PLUGIN NAME

Plugin Name: PDF Embedder
Plugin URI: https://github.com/franciscocervera/PDF-Embedder
Description: Embeds PDF files to your WordPress pages/posts using browser's default viewer.
Version: 0.1
Author: Francisco Cervera
Author URI: https://www.github.com/franciscocervera

DESCRIPTION

Embeds PDF files to your WordPress pages/posts using HTML embed tag.

INSTALLATION

1. Upload PDF Embedder folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place "<?php do_action('plugin_name_hook'); ?>" in your templates (If necessary).

USAGE

1. Add a PDF file from the media library. 
2. A shortcode will be generated automatically
3. Once published, the file will be automatically displayed in the browser.
